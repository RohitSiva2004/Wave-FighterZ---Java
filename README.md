# Dragon Ball Z Inspired JavaFX and Java Game

## Overview

Dive into an action-packed adventure inspired by Dragon Ball Z in this JavaFX and Java game. Navigate through multiple maps, battle waves of enemies, and unleash powerful attacks as Goku. With immersive gameplay mechanics, dynamic character transformations, and strategic power-ups, this game promises an exhilarating experience for fans of the iconic anime.

## Main Features

### Title Screen

- **PLAY**: Begin your journey by selecting this option, leading you to the Map Select screen.
- **CONTROLS**: Familiarize yourself with the game controls:
  - W: Move Up
  - A: Move Left
  - S: Move Down
  - D: Move Right
  - Space: Shoot Ki Blast Projectiles
- **HIGH SCORE**: View the highest scored player, saved in a text file.
- **EXIT**: Quit the game.

### Map Select Screen

Choose from four distinct maps to begin your battle. One map is randomly selected each time you play, ensuring a unique experience.

## Gameplay

- **Character**: Start as Goku in his base form, defending against waves of enemies.
- **Enemies**: Face different types of monsters, each requiring a specific number of ki blasts to defeat:
  - Standard Monster: 1 hit
  - Wave 5 Enemy: 2 hits
  - Wave 10 Enemy: 3 hits
- **Health**: Monitor Goku's health bar displayed at the top of the screen. If a monster hits Goku or reaches the left side of the screen, Goku's health decreases.
- **Power-Ups**:
  - Heart: Restores a percentage of health.
  - Dragon Ball: Transforms Goku into Super Saiyan, granting temporary invincibility and rapid ki blast shooting.
  - Star: Provides brief invincibility.

## High Scores and Game Over

- **High Scores**: After each game, the highest wave number reached is saved in a text file. This score is displayed on the high score screen.
- **Game Over Screen**: When Goku's health reaches 0, the game ends. Players can choose to return to the main menu or quit the game.

## Dynamic Character Model

Goku's character model updates based on his form:

- Base Form
- Super Saiyan: When transformed by a Dragon Ball power-up.
- Ki Blast Shooting
- Death Animation

## Audio

- **Ki Blasts**: Sound effects for shooting.
- **Super Saiyan Transformation**: Unique audio cues.
- **Background Music**: Continuous music playing throughout the game.

This Dragon Ball Z inspired game brings the thrill and excitement of the anime to your screen. Whether you’re a long-time fan or a newcomer, the blend of strategic gameplay, character transformations, and engaging audio will keep you hooked. Download now and see how many waves you can survive!
